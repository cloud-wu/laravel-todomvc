# laravel-todomvc

## Quck Installation

    git clone https://github.com/cloud-wu/laravel-todomvc.git

    cd laravel-todomvc

    cp .env.example .env
    
    composer install
    
    php artisan migrate
    
    npm install
    
    npm run dev
